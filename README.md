# <p align="center">TicTacToe</p>

## Objectif
Concevoir une application avec Vite.js et React.js.
L'application permettra de jouer à un TicTacToe.

## 🧑🏻‍💻 Pour démarer le projet
```bash
npm i
```
```bash
npm run dev
```
